Weather App
____________

Build a web application that allows the user to enter a city name and retrieve the current weather information.

1 – Geocoding API (Convert City >> Coordinates)

When the user clicks the "Get Weather" button:

Send a request to the following API to convert the city name into latitude and longitude:

https://geocoding-api.open-meteo.com/v1/search?name=CITY_NAME

Replace CITY_NAME with the value entered by the user.

From the response, extract:

latitude

longitude

name

country


2 – Weather API (Get Current Weather)

After retrieving the coordinates, send another request to:

https://api.open-meteo.com/v1/forecast?latitude=LAT&longitude=LON&current_weather=true

Replace:

LAT with latitude

LON with longitude

From the response, extract:

temperature

windspeed

time

3- then finally you should Display the Following Information:


City Name

Country

Temperature (°C)

Wind Speed (km/h)

Weather Time
